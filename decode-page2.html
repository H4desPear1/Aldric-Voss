Cipher Analysis Workbench

Substitution cipher editor · Mirror / reversal analysis · Frequency counting — IESR Internal Tool v1.8
About This Tool

This workbench is designed for analysis of historical substitution cipher systems in manuscript contexts. It is currently configured to support work on the Codex Noctis (Folio 2v — Page 2), which contains an unresolved cipher line (Line 1) followed by text in partially deciphered Elder Futhark.

Codex Noctis — Page 2 context: Line 1 of Folio 2v presents 9 unknown glyphs: ᛜ · ᛡ · ᚣ · ᛠ · ᛟ · ᛝ · ᚸ · ᛢ · ᛣ. Voss believed this line functions as a substitution key — each cipher glyph maps to a standard character value. Once the mapping is established, the full text of Page 2 can be decoded. The theme of the remaining text (a mirror metaphor, a face on the other side) may itself be a clue to the cipher method.
Mirror / Reversal Analysis

Some historical ciphers use reversed or mirrored character sequences. Enter a string to view forward, reversed, and mirror-reversed outputs.

Hint (Page 2): The text of Folio 2v uses the metaphor of a mirror repeatedly — "on the other side of the mirror lies the true face." If the cipher glyphs in Line 1 are themselves mirror-reversed or positionally inverted versions of standard runes, the substitution key may be derived by applying a reversal transformation. Try applying reversal to known Elder Futhark characters and compare to the cipher glyphs in Line 1.
Custom Substitution Cipher — Key Builder

Define a custom glyph-to-value mapping to apply to ciphered text. Enter your proposed character values for each cipher glyph from Line 1 of Folio 2v, then use the decode field below to apply your key.

Output will appear here once key is applied
Frequency Counter

Count character or token frequencies in any input text. Useful for identifying common letters that may anchor a substitution key.

Page 2 — Full Transliterated Text (Voss Draft)

Below is Dr. Voss's working transliteration of Folio 2v (excluding the unresolved cipher line), provided for reference. The readable sections translate approximately as: "On the other side of the mirror lies the true face / you see now only your mirror image / look into the mirror — never / or the mirror will fall / three names bear the truth / the dead man's name is one / the killer's name is two / the witness's name is three / name all three aloud."

Key question: Lines 9–12 name three roles: the dead, the killer, the witness. Page 1 instructed the reader to "speak the name." Page 2 instructs: name all three aloud. The cipher line (Line 1) may encode the method for deriving those names. Solve the cipher glyphs using the mirror/reversal tool and the key builder above.
